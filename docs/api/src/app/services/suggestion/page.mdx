# Suggestions

As the name suggests, the Suggestions API allows to suggest posts to the user based on their interests. This is a great way to increase user engagement and retention. The API provides a number of endpoints to manage suggestions programmatically.



### Properties

<Properties>
  <Property name="events" type="array">
    An array of events that the user has interacted with.
  </Property>
  <Property name="interests" type="array">
    An array of interests that the user has.
  </Property>
</Properties>

---

## List all contacts {{ tag: 'POST', label: '/v1/suggestion' }}


<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your suggestions based on the provided query parameters.

  </Col>

  <Col sticky>

    <CodeGroup title="Request" tag="POST" label="/v1/suggestion">

      ```json
      {
        "events": [
          {
            "id": "1",
            "content": "This is a test event",
          },
          {
            "id": "2",
            "content": "This is another test event",
          }
        ],
        "interests": [
          {
            "id": "1",
            "content": "Bitcoin",
          },
          {
            "id": "2",
            "content": "Future",
          }
        ]
      }
      ```

    </CodeGroup>

    ```json {{title: 'Response'}}
    {
      "data": []
    }
    ```

  </Col>
